Discuss the scope of var, let, and const ?

var: Variables declared with var have function scope or global scope.
let: Variables declared with let have block scope. They are limited to the block or statement in which they are defined.
const: Variables declared with const also have block scope like let.

Tell us the use cases of null and undefined ?

null: Explicitly represents the absence of a value. Used when you want to indicate that a variable or property intentionally has no value.
undefined: Represents the absence of a value when a variable has been declared but hasn't been assigned a value.Automatically assigned to variables that have been declared but not initialized.

What do you mean by REST API ?

REST API stands for "Representational State Transfer Application Programming Interface." It is a set of conventions and principles for building and interacting with web services. REST APIs are designed to be simple and scalable, allowing different software applications to communicate with each other over the internet.
